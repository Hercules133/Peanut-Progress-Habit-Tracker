darkMode

darkMode top-level property

ThemeData darkMode
getter/setter pair

Color and design theme for dark mode. This theme is used in the app when the device is in dark mode or when the user has selected light mode in the app settings. The theme is based on the Material Design color scheme and the custom colors defined in the OwnColors class.

Implementation
