Retrieves an item from the storage based on the specified table and key.

table - The name of the table to search in. key - The key used to identify the item in the table.

Returns a Map<String, dynamic> representing the stored item, or null ifq the item is not found.

Implementation

@override
Future<Map<String, dynamic>?> read(String table, String key) async {
  final prefs = await _getPrefs();
  final List<String> items = prefs.getStringList(table) ?? [];

  for (var item in items) {
    final Map<String, dynamic> decodedItem = jsonDecode(item);

    if (decodedItem.containsKey(key)) {
      return decodedItem;
    }
  }

  return null;
}
